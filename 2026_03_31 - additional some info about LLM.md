### 주요 문자 기호
하이픈(-), 목록 작성
- 목록의 각 항목을 표시하는 데 사용

따옴표(“”)
- 특정 단어나 문구를 강조하거나, 모델이 정확하게 인식해야 하는 텍스트를 표시할 때 사용

콜론(:)
- 목록이나 설명을 시작할 때 사용

별표(*)
- 목록을 나타내거나 특정 내용을 강조할 때 사용

대괄호([])
- 추가 정보나 선택 사항을 나타낼 때 사용

중괄호({})
- 특정 조건을 명시할 때 사용

세 개의 백틱(```)을 앞 뒤로 사용
- 코드 블록을 표시하는 데 사용, 프롬프트 내에서 코드나 명령어를 명확하게 구분

해시 기호(#)
- 마크다운(Markdown) 문법에서 제목이나 섹션을 구분하는 데 사용
- 프롬프트를 구조화하여 모델이 각 부분을 명확하게 이해하도록 도움

### 대화 질문 설계 요소: 키워드, 제한, 어조, 부정

키워드와 제한
- 핵심 키워드를 정확하게 사용하는 것이 중요
    - 친구에게 맛집을 물어볼 때 “매운 떡볶이 맛집”이라고 구체적으로 말하는 것과 같은 원리
    - “한국 역사”보다는 “조선시대 세종대왕 업적”이라고 질문
- 답변의 범위나 형식을 제한하는 것도 유용
    - “100자 이내로 요약해 줘” 또는 “표 형식으로 정리해 줘”와 같은 조건을 추가

어조와 부정
- 사용자가 요청 시 AI가 더 부드럽고 친절한 어조로 답변할 가능성이 높음
    - “이 주제에 대해 설명해 줘”보다는 “이 주제에 대해 친절하게 설명해 줘”
- 부정적인 표현을 사용하여 원치 않는 결과를 제외
    - “인공지능의 장점만 알려줘”보다는 “인공지능의 단점은 제외하고 장점만 알려줘”

### 퓨샷 러닝(Few-shot learning)과 퓨샷 프롬프팅(Few-shot prompting)

퓨샷 러닝(FSL: Few-shot learning)
- AI 모델이 제한된 수의 예시만으로 새로운 개념이나 작업을 학습하는 기술
    - 어린 아이가 몇 번의 경험으로 새로운 단어나 사물을 이해하는 방식과 유사
    - 적은 데이터로 높은 학습 효율을 보이는 놀라운 능력

- 기존의 머신 러닝 모델은 방대한 양의 데이터를 필요하지만, 퓨샷 러닝은 단 몇 장의 고양이 사진만으로도 새로운 고양이를 인식
    - 데이터 부족: 새로운 분야나 희귀한 데이터의 경우, 충분한 학습 데이터를 확보하기 어려움
    - 빠른 적응: 새로운 환경이나 작업에 신속하게 적응해야 할 때
    - 개인화: 개인의 취향이나 특성에 맞춰 모델을 학습시킬 때


퓨샷 프롬프팅(few-shot prompting)
- 퓨샷 러닝의 능력을 극대화하여 우리가 원하는 결과를 더욱 정확하게 얻을 수 있도록 도와주는 마법 같은 기술
    - AI가 단순히 학습하는 것을 넘어서, 우리가 원하는 결과를 ‘콕’ 집어내도록 도와줄 수 있음

### 제로샷 프롬프팅
- AI에게 아무런 예시 없이 바로 원하는 결과를 요청하는 방법
    - 처음 보는 외국어를 번역가에게 요청하는 것과 유사
- AI는 방대한 지식을 바탕으로 질문에 답하며, 
    - 예를 들어 “에펠탑에 대해 설명해 줘”라고 하면 역사, 구조, 특징 등을 설명
- 예시
    - 다음 문장의 감정을 분류해 주세요: '오늘 옷을 사서 돈을 많이 지출했어요. 
    - https://url.kr/uwneqv

### 원샷과 투샷 프롬프팅 (https://url.kr/uwneqv)

- 각각 하나 또는 두 개의 예시를 제공하여 원하는 결과를 유도하는 방식
- 친구에게 새로운 게임 방법을 한두 번 시범 보이는 것
    - 예를 들어, “긍정 적인 영화 리뷰: 정말 감동적이었어! / 부정적인 영화 리뷰: 너무 지루했어.” 와 같은 예시를 통해 새로운 리뷰의 감정을 판단하도록 할 수 있음
- 제로샷보다 정확도가 높고, 간단 한 패턴을 학습하는 데 효과적


---

### 퓨샷 프롬프팅
- 여러 개의 예시를 제공하여 원하는 결과를 유도하는 방식
- 요리 선생님에게 여러 번 레시피를 배우는 것과 유사
    - 예를 들어, 다양한 과일 사진과 이름을 보여주고 새로운 과일 사진을 맞추도록 할 수 있음
- 퓨샷 프롬프팅은 복잡한 패턴이나 규칙을 학습하고 다양한 맥락을 이해하는 데 효과적
