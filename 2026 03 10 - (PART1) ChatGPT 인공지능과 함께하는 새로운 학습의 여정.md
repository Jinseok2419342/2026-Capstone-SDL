## 생성형  AI

(기존 데이터를 바탕으로 새로운 콘텐츠를 만들어내는 인공지능 기술)

- 여러 매체에서 활용

- 생성형 AI의 주요 특징
    - 콘텐츠 생성 - 주어진 입력에 따라 새로운 텍스트, 이미지, 음악 등을 생성
        - 사용자가 특정 주제를 입력하면 그에 맞는 글을 작성
        - 특정 스타일의 이미지를 생성
    - 창의성
        - 기존의 데이터를 학습하여 새로운 아이디어나 콘텐츠를 만들어내는 능력을 가짐 (예술, 디자인, 마케팅 등 다양한 분야에서 활용)
    - 응용 분야
        - 게임, 영화, 광고, 교육 등 여러 산업에서 사용 (사용자 맞춤형 콘텐츠를 제공하는 데 큰 기여)

## 거대 언어 모델 (LLM)

(자연어 처리(NLP: Natural Language Processing) 분야에서 활용되는 특별한 유형의 생성형 AI)

인공지능 기술의 중요한 축을 구성 

간단히 LLM(Large Language Model)

- 방대한 양의 텍스트 데이터를 학습하여 언어의 문맥을 이해하고 의미 있는 응답을 생성하는 데 뛰어난 능력을 가짐

거대 언어 모델의 주요 특징

- 언어 이해
    - 문맥을 파악하고 문법적으로 올바른 문장을 생성하는 능력을 갖추고 있음 (대화형 AI, 번역기, 텍스트 요약기 등 다양한 응용 프로그램에서 널리 활용)
- 대화 생성
    - 사용자의 질문이나 요청에 대해 자연스럽고 일관된 대화를 만들어 낼 수 있음
        - 이는 고객 서비스 챗봇이나 개인 비서와 같은 분야에서 특히 유용하게 사용
- 지식 기반
    - 학습한 데이터를 바탕으로 다양한 주제에 대한 정보를 제공 가능
        - 특정 주제에 대한 질문에 대한 답변을 생성하는 데 매우 강점

## ChatGPT

(2015년 설립된 OpenAI는 2018년, ChatGPT라는 거대 언어 모델의 가능성을 세상에 제시)

거대 언어 모델의 대중화

- 압도적 성능과 사용자 친화 인터페이스
- 광범위한 지식과 정보를 채팅이라는 친근한 대화 형식으로 제공

GPT(Generative Pre-trained Transformer) 모델을 기반

- 사전 학습된 방대한 데이터를 활용
    - 주로 사용자 질문에 답변
    - 문제 해결, 정보 서치, 창작 아이디어 제공 등 다양한 분야에서 사용되고 있음

## 기술 용어

GPT(Generative Pre-trained Transformer)

- GPT는 'Generative Pre-trained Transformer'의 약자로서 번역하면 '생성 사전 훈련 변환기'라고 할 수 있습니다. GPT는 마치 똑똑한 글쓰기 친구와 같아요. 인터넷에 있는 수많은 글을 읽고 학습하여, 어떤 주제든 자연스럽게 글을 쓸 수 있죠. GPT는 문장을 이해하고 다음 단어를 예측하는 능력이 뛰어나, 마치 사람이 쓴 것처럼 자연스러운 글을 만들어냅니다. 예를 들어, GPT에게 "고양이"에 대한 글을 써달라고 하면, 고양이에 대한 재미있는 이야기를 술술 써 내려가는 거죠.

사전 훈련(Pre-training)

- 사전 훈련은 GPT가 책을 통해 세상을 배우는 과정과 같아요. 인터넷에 있는 수많은 텍스트 데이터를 읽 으면서 단어와 문장의 관계, 문법, 다양한 지식을 익히는 거죠. 마치 어린아이가 책을 읽으며 세상을 배워 나가는 것처럼요. 이렇게 사전 훈련을 통해 GPT는 방대한 지식을 습득하고, 다양한 주제에 대한 글을 쓸 수 있는 능력을 갖추게 됩니다.

미세 조정(Fine-tuning)

- 미세 조정은 사전 훈련을 마친 GPT에게 맞춤형 과외 수업을 하는 것과 같아요. 특정 분야의 데이터를 추
가로 학습시켜 원하는 결과물을 더 잘 만들어내도록 하는 거죠. 예를 들어, GPT를 시인으로 만들고 싶다 면 시를 집중적으로 학습시키고, 번역가로 만들고 싶다면 번역 데이터를 학습시키는 거예요. 이렇게 미세 조정을 통해 GPT는 특정 분야에 특화된 능력을 갖추게 됩니다.

## 역대 GPT 초기 모델 역사

GPT-1 (2018년)

- 방대한 텍스트 데이터를 바탕으로 사전 훈련과 미세 조정을 통해 자연스럽게 문장을 생성

GPT-2 (2019년)

- 모델의 크기와 성능이 비약적으로 향상
- 특정 주제에 대해 논리적인 문장을 생성하는 능력이 뛰어나, 사람과 유사한 텍스트를 생성
- GPT-2의 악용 가능성을 우려하여 초기에는 모델을 공개하지 않고, 점진적으로 연구자들에게 개방하는 방식으로 접근

GPT-3 (2020년)

- 본격적인 실용 AI 모델
- GPT-3는 1,750억 개의 매개변수를 가진 거대 언어 모델
- 이전 모델에 비해 훨씬 더 정교한 언어 이해 및 생성 능력 보유

## 다양한 생성 분야 활용

- 업무 생산성 향상은 거대 언어 모델의 가장 두드러진 장점
    - 예를 들어, 고객 지원 분야에서 거대 언어 모델은 실시간으로 질문에 답하고 문제를 해결하여 고객
    만족도를 향상
    - 콘텐츠 생성 및 마케팅 분야에서도 창 의적인 아이디어를 제공하고, 빠른 시간 내에 고품질 콘텐츠
    를 생산하게 해 기업의 운영 효율 성과 경쟁력을 강화

- 개인의 능력 혁신 측면에서도 거대 언어 모델은 중요한 역할
    - 학습 도구로서 사용 자 맞춤형 교육을 제공
    - 개인의 학습 속도와 스타일에 맞춘 자료를 제시하여 학습 효율성 을 높이며 전문성을 강화

- 거대 언어 모델은 창의적인 작업의 협업으로 혁신적 결과물 작성에 기여

    향상

1. AI 언어 모델은 인간 수준의 언어 이해 및 생성 능력을 달성할 것
    - 방대한 데이터를 학습해 문맥을 깊이 이해하고, 창의적인 글쓰기나 복잡한 문제 해결 등 다

2. 로봇공학과 AI 기술의 융합으로 인간과 유사한 로봇들이 우리의 일상에 점점 더 많이 도입
    - 로봇들은 가사 노동, 간병, 서비스 산업 등에서 인간을 보조하거나 대 체하여 삶의 질을 향
    상시키고 효율성을 향상
3. AI 기술의 수요 증가로 AI 칩 시장의 경쟁이 심화
    - 여러 기업들이 AI 칩 개발에 참여하며, 이는 공급 증가와 가격 하락, 기술 발전
4. AI는 의료, 교육, 금융 등 다양한 분야에서 혁신을 주도하며, 생산성 향상과 새로운 비즈니스
모델 창출
    - 그러나 이러한 변화는 일자리 구조와 윤리적 문제 등 새로운 사회적 과제를 동반
5. 마지막으로, AI 발전과 함께 윤리적 고려와 규제의 중요성이 대두됨
    - 투명성, 책임성, 공정성을 확보하기 위한 정책 마련 필요
    - AI 기술의 신뢰성과 수용성을 높이는 데 필수
