### GPT-1 (2018년)
- 방대한 텍스트 데이터를 바탕으로 사전 훈련과 미세 조정을 통해 자연스럽게 문장을 생성

### GPT-2 (2019년)
- 모델의 크기와 성능이 비약적으로 향상
- 특정 주제에 대해 논리적인 문장을 생성하는 능력이 뛰어나, 사람과 유사한 텍스트를 생성
- GPT-2의 악용 가능성을 우려하여 초기에는 모델 을 공개하지 않고, 점진적으로 연구자들에게 개방
하는 방식으로 접근

### GPT-3 (2020년)
- 본격적인 실용 AI 모델
- GPT-3는 1,750억 개 의 매개변수를 가진 거대 언어 모델
- 이전 모델에 비해 훨씬 더 정교한 언어 이해 및 생성 능력 보유

### ChatGPT-1.0 (2022년)
- ChatGPT는 GPT-3.5 기반의 챗봇(chatbot)으로, 사용자와 자연 스러운 대화를 나누도록 특별히 최
적화된 모델
- 인간 피드백 기반 강화 학습(RLHF: Reinforcement Learning from Human Feedback) 기술을 도
입해 성능을 향상
    - 인간의 피드백을 통해 인공지능 모델이 학습하고 성능을 개선하는 방법
- ChatGPT는 단순한 대화형 AI를 넘어 코딩, 작문, 번역 등 다양한 작업을 지원하는 강력한 도구

### 활용 분야
업무 생산성 향상은 거대 언어 모델의 가장 두드러진 장점
- 예를 들어, 고객 지원 분야에서 거대 언어 모델은 실시간으로 질문에 답하고 문제를 해결하여 고객
만족도를 향상
- 콘텐츠 생성 및 마케팅 분야에서도 창 의적인 아이디어를 제공하고, 빠른 시간 내에 고품질 콘텐츠
를 생산하게 해 기업의 운영 효율 성과 경쟁력을 강화

개인의 능력 혁신 측면에서도 거대 언어 모델은 중요한 역할
- 학습 도구로서 사용 자 맞춤형 교육을 제공
- 개인의 학습 속도와 스타일에 맞춘 자료를 제시하여 학습 효율성 을 높이며 전문성을 강화

거대 언어 모델은 창의적인 작업의 협업으로 혁신적 결과물 작성에 기여
